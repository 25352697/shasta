Prerequisite packages

To download, build and run the shasta software on a ubuntu 16.04 or 18.04 machine, the following packages are required:

  • git
  • g++
  • cmake
  • The Boost libraries
  • Graphviz
  • Python 3
  • pip3
  • pybind11
  • The png library
  • hugepages
  • SeqAn 2.4.0

Script shasta/scripts/InstallPrerequisites-Ubuntu16.04.sh can be used to install these prerequisites. It must be run with root privileges. After building the code, this script is also available as shasta-install/bin/InstallPrerequisites-Ubuntu16.04.sh.

Note that SeqAn 2.4.0 is not available on standard Ubuntu repositories, and shasta/scripts/InstallPrerequisites-Ubuntu16.04.sh downloads it from GitHub. See comments in the script for more information.
